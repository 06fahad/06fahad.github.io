    Some Of My Projects

    • YelpCamp

      YelpCamp is a web app where users can post about new campgrounds or leave reviews/comments about existing campgrounds that they have been to. Front-end built using bootstrap and back-end built on NodeJS, deployed using Heroku. NoSQL database implemented using MongoDB to maintain information about campgrounds, users, reviews, comments etc. Although still in development, you can check out the website here.

    • Mapper103

      One of the larger projects that I have participated in, the Mapper103 is a geographcial information system. The application is built using C++ and uses openGL and the OpenStreetMaps API to create a functional mapping application. User is able to view map of any city, pan and zoom on the map, find shortest path from chosen location to destination, find the closest gas station/Starbucks etc. You can check out all the code on my github here.

    • Patatap Clone

      Patatap is a portable animation and sound kit. I created a slightly dumbed down version of it using howler.js and paper.js. Check it out here. This is a front-end only app that I did just for fun so I haven't bothered to deploy it. You can easily run it on your browser by downloading the files from the git repo and opening the HTML file.

    • To-Do List Web App

      A simple to-do list web app using jQuery that I developed in my spare time just for fun. You can check it out on my github, download the files and run it on your browser by opening the HTML file on your browser.
